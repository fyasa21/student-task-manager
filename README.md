# Student Task Manager

## 📌 Deskripsi
Student Task Manager adalah aplikasi sederhana untuk membantu mahasiswa mencatat dan mengelola tugas kuliah. Aplikasi ini dibuat sebagai proyek praktikum mata kuliah Rancang Bangun Perangkat Lunak dengan menerapkan version control Git secara kolaboratif.

---

## 🎯 Tujuan
- Memahami penggunaan Git dalam pengembangan software
- Melatih kerja tim menggunakan sistem version control
- Mengimplementasikan konsep IPO (Input, Process, Output)

---

## ⚙️ Fitur
- Tambah tugas
- Edit tugas
- Hapus tugas
- Tandai tugas selesai
- Urutkan berdasarkan deadline

---

## 🧠 Konsep IPO

**Input**
- Judul tugas
- Mata kuliah
- Deadline
- Status

**Process**
- Validasi data
- Sorting deadline
- Update status

**Output**
- Daftar tugas
- Status tugas
- Informasi deadline terdekat

---

## 📁 Struktur Project
- docs/ → dokumentasi
- src/ → source code
- tests/ → unit testing

## ▶ Cara Menjalankan
1. Clone repository
2. Masuk folder project
3. Jalankan program sesuai bahasa yang digunakan
